### ¿Qué hace el plugin?

Es altamente configurable, y es una herramienta poderosa y rápida para administrar los grief (rollbacks, restores, etc) en tu servidor.

## Usage

:::important
Antes de iniciar, necesitas haber creado una base de datos MySQL. Puedes encontrar más información [aquí](https://docs.bloom.host/languages/spanish/basico/panel/databases).
:::


[Descarga el plugin](https://jenkins.addstar.com.au/job/Prism-Bukkit%201.16/), y sube el jar dentro de tu carpeta `plugins`, y reinicia el servidor. Si necesitas ayuda instalando plugins, puedes leer la guía de [Cómo instalar plugins](https://docs.bloom.host/languages/spanish/basico/instalar-plugins). 

Luego de ello, entra a la carpeta `Prism`, la cual puedes encontrar dentro de la carpeta `plugins`. Desde ahí, edita el archivo `config.yml`. 

Baja hasta la sección `mysql`, y entra los datos de tu base de datos. También cambia `useNonStandardSql:true` a `useNonStandardSql: false`.

:::caution
No copies la configuración de abajo de forma exacta, tienes que ingresar los datos de tu propia base de datos, la cual es diferente. `
:::

```YML
  mysql:
    hostname: 135.181.0.179
    username: u398_a63h3x4IdH
    password: 'BMMQhZ@2oCt4PYDs02LS^1^m'
    databaseName: s398_minecraft
    prefix: prism_
    port: '3306'
    useNonStandardSql: false
```
Una vez terminaste eso, reinicia el servidor para que se apliquen los cambios.
